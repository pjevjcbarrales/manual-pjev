Estándar de Reportes Simples (JavaScript)

Reglas para generar reportes tabulares, catálogos e impresiones base directamente desde el cliente (Frontend) utilizando librerías de JS.

1 ¿Cuándo usar un Reporte Simple?

Se consideran "Reportes Simples" aquellos que cumplen con las siguientes características:

  • Consisten principalmente en listados, tablas o catálogos (ej. Catálogo de Cuentas, Listado de Proveedores).
  • No requieren diseños complejos de impresión (como folios pre-impresos, acomodo exacto de membretes, marcas de agua complejas, pólizas o afectaciones de gasto).
  • La generación ocurre mayoritariamente en el cliente, apoyándose en la información que el Frontend recibe en formato JSON o directamente desde el servidor como string HTML/CSV.

2 Formatos de Salida Soportados

Todo reporte simple generado para un catálogo debe proveer al usuario la capacidad de elegir criterios (filtros) y el formato de exportación mediante un modal wizard estándar. Los formatos obligatorios son:

Formato Descripción Método de Generación
PDF Visualizar en pantalla e imprimir nativamente. Se genera una plantilla HTML a partir de los datos JSON y se utiliza window.open para imprimir nativamente o se usa una librería de jspdf para pintar el documento desde el Frontend.
Excel (.xls / .xlsx) Archivo compatible con Microsoft Excel. Generado vía librerías JavaScript que transforman el JSON a binario (ej. xlsx o `exceljs`).
CSV Valores separados por coma (Codificación UTF-8). Generación de un Blob en texto plano separando los campos con comas e inyectando BOM para que Excel respete los acentos.

// 1. Componente Modal de Selección (Wizard)
<ModalImprimir>
  - Paso 1: Seleccionar Criterios (Filtros de búsqueda o estructura)
  - Paso 2: Seleccionar Formato (PDF, Excel, CSV)
</ModalImprimir>

// 2. Ejecución desde la vista principal de la página:
const res = await fetch(`/api/.../reporte?formato=${formato}`);

// 3. Lógica de renderizado según el formato elegido:
if (formato === 'pdf') {
  setReporteData(res.data); // Abre el <ReporteVisor /> HTML que vemos en el Mockup
} else if (formato === 'csv') {
  descargarCSV(res.data);   // Dispara archivo descargable
} else {
  descargarExcel(res.data); // Dispara archivo descargable
}
