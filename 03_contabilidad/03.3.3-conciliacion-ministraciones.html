03.3.3

Conciliación de Ministraciones (Ingresos SEFIPLAN)

Análisis Funcional bajo el Estándar Oficial DOC-08 del SIAF-PJEV.

Tipo: Herramienta de Cierre / Validaciones Cruzadas
1 Descripción General

El Poder Judicial del Estado de Veracruz recibe su presupuesto (subsidio estatal) mediante transferencias periódicas llamadas "Ministraciones" emitidas por la Secretaría de Finanzas y Planeación (SEFIPLAN). Este módulo es vital para asegurar que el ingreso presupuestal (Ley de Ingresos) y el ingreso contable (Depósitos en Bancos) cuadren al centavo con el calendario de pagos oficial autorizado por el Estado.

Definición SQL (PostgreSQL)

-- ==========================================
-- 1. LEER CONCILIACIÓN DE MINISTRACIONES
-- ==========================================
CREATE OR REPLACE FUNCTION contabilidad.fn_leer_conciliacion_ministraciones(
    p_ejercicio INT
) RETURNS jsonb LANGUAGE plpgsql AS $$
DECLARE v_resultado jsonb;
BEGIN
    SELECT COALESCE(jsonb_agg(row_to_json(t)), '[]'::jsonb) INTO v_resultado
    FROM (
        SELECT 
            m.mes,
            m.monto_autorizado,
            COALESCE(r.monto_recaudado, 0) AS monto_recaudado,
            (m.monto_autorizado - COALESCE(r.monto_recaudado, 0)) AS diferencia,
            m.observaciones
        FROM ingresos.ing_ministraciones m
        LEFT JOIN (
            -- Subconsulta que suma las pólizas de recaudo agrupadas por mes
            SELECT EXTRACT(MONTH FROM i.fecha_deposito) AS mes_rec, SUM(i.monto_total) AS monto_recaudado
            FROM ingresos.ing_ingresos i
            WHERE i.ejercicio = p_ejercicio AND i.estatus_documento = 4
            GROUP BY 1
        ) r ON m.mes = r.mes_rec
        WHERE m.ejercicio = p_ejercicio
        ORDER BY m.mes
    ) t;
    
    RETURN jsonb_build_object('cod_resultado', 1, 'msg_resultado', 'OK', 'data', v_resultado);
END;
$$;
Parámetros (IN)TipoDescripción
@p_ministracion_idINTID de la ministración.
@p_observacionesTEXTJustificación.
Valores Devueltos (OUT) JSONTipoDescripción

Definición SQL (PostgreSQL)

-- ==========================================
-- 2. GUARDAR OBSERVACIONES
-- ==========================================
CREATE OR REPLACE FUNCTION contabilidad.fn_guardar_observaciones_ministracion(
    p_ministracion_id INT,
    p_observaciones TEXT
) RETURNS jsonb LANGUAGE plpgsql AS $$
BEGIN
    UPDATE ingresos.ing_ministraciones
    SET observaciones = p_observaciones,
        fecha_m = CURRENT_TIMESTAMP
    WHERE ministracion_id = p_ministracion_id;
    
    RETURN jsonb_build_object('cod_resultado', 1, 'msg_resultado', 'Observación guardada');
END;
$$;

7 Acciones del Módulo

El sistema expone las siguientes acciones operativas:

  • Vista de Semáforos: El grid principal colorea en Verde los meses cuadrados y en Rojo/Naranja los meses donde llegó menos dinero del pactado.
  • Anotaciones de Auditoría: Permite escribir textos justificativos (Ej. "SEFIPLAN aplicó retención directa de ISR por 5 MDP, de ahí la diferencia").
  • Impresión / Exportación: Botón 🖨️ que genera el "Reporte Analítico de Ministraciones" exigido para la Cuenta Pública Trimestral.

10 Flujos de Integración

Es un módulo de monitoreo. Hacia atrás, se alimenta del Módulo de Presupuesto de Ingresos (Calendario) y del Registro de Pólizas de Ingreso (Bancos). Hacia adelante, el reporte impreso aquí es requisito indispensable para armar el legajo físico de los Estados Financieros y Cuenta Pública Trimestral.

PROCESO / ÁREA DIRECCIÓN DE FINANZAS SIAF-PJEV
1. CONSULTA Ingresa al módulo a fin de mes para verificar si SEFIPLAN radicó el total del dinero correspondiente al mes actual. Agrupa el total de pólizas de ingreso con cuenta `8150` de ese mes y las compara matemáticamente contra la tabla de `presupuesto_calendario`. Pinta semáforos.
2. JUSTIFICACIÓN (Si Aplica) Si el semáforo es rojo (Diferencia), el Director redacta la explicación oficial solicitada por el ORFIS. Guarda el texto en base de datos vinculado al registro mensual.
3. GENERACIÓN DE REPORTES Presiona Imprimir para anexar al expediente de Cuenta Pública Trimestral. Exporta matriz completa con la suma "A la fecha" e imprime las notas al pie.
